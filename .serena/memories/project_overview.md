# Project Overview

- Project: `llm-wiki` / WikiMind.
- Purpose: local-first personal/small-team knowledge base where raw materials are immutable and agents collaborate on a Markdown wiki layer with traceability and linting.
- Current authoritative product/design docs live under `spec-v2/`; current implementation is a Go codebase scaffold with CLI/daemon entrypoints and internal packages.
- Main roadmap source for implementation work: `spec-v2/docs/roadmap-30d.md`.
- Trellis task/spec workflow lives under `.trellis/`; active coding tasks should follow Trellis skills and `.trellis/spec/` guidance.
